# Python venv in win10 wsl2
Versions

- Ubuntu 20.04.2 LTS
- Python 3.8.5
***
**commands for venv**
``` bash
# create venv
python3 -m venv .venv

# activate venv
source .venv/bin/activate 

# shows python version
which python

# deactivate venv
deactive
```
**commands to dependensy management**
```bash
# installs dependen
pip freeze > requirements.txt

# installs dependen
pip install -r requirements.txt
```
